HTML

  • HTML forms the structure of a webpage.
  • The head element contains information (metadata) about the webpage.
    • The style element defines the CSS styles associated with a webpage.
    • The base element defines the base URL for a webpage.
    • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • The body element represents the visible content shown to the user.
    • The header element appears at the top of the page and usually contains a navigation menu and introductory content.
    • The footer element appears at the bottom of the page and usually displays the author, copyright and contact information, sitemap, and navigation.
    • The ul, ol, and li elements represent unordered lists, ordered lists, and list items.
    • The img element contains information about images displayed on the webpage. The src attribute defines the location of the image.
    • The br element creates a line break.
    • The script element embeds client-side scripts in the HTML document.

CSS

  • CSS adds style to the webpage.
  • Declarations contain two components: a property and a value.
  • The * selector applies a rule to all visible elements on the page.
  • margin indicates how much space we want around the outside of an element.
  • padding adds space around the content inside an element.
  • Use a preceding . to indicate a class selector.

JavaScript

  • JavaScript adds functionality and interactivity to the webpage.
  • Use the console.log() function to print something to the console.
  • Use the var keyword to declare and assign (=) a variable.
  • Control flow is the order in which a computer executes code in a script. In JS, code runs from the first statement at the top of the page, down to the last statement at the bottom (unless interrupted by conditional logic or loops).
  • A value is considered falsy in JS if it is one of the following values: 0, -0, "", On, null, undefined, NaN, or a boolean data type with a value of false. All other values are truthy.
  • In a if statement, the computer goes through all listed conditions, evaluating whether or not a condition is truthy. Once it finds a condition that is truthy, it runs the appropriate code contained in curly brackets then breaks out of the if statement.
  • Arrays in JS are zero-indexed and sequential.
  • for loops can repeat a block a code, considering an initial state, a condition for executing the code block, and an update expression.
  • A function in JS is a set of instructions that tells the computer how to perform a specific task. Functions must be called by name for it to execute.
  • A function's name is always followed by a set of parentheses that usually contain parameters for the function to consider.
  • Use functions and loops to eliminate repetitive code.

Git

  • Use Git and a cloud-based repository like GitHub to store your code and collaborate with others.
  • Commonly used commands:
    • Git is the most commonly used version control software among developers.
    • git status: checks what branch we are currently on.
    • git checkout -b branch-name: creates a new branch and switches to it.
    • git add -A: adds all local changes to the staging area.
    • git commit -m "commit message...": commits staged changes and associates a commit message.
    • git pull origin main: pulls changes for the remote repository.
    • git push origin branch-name: pushes local changes to the remote repository.
